# Financial Dashboard (Next.js + Tailwind)

A clean, modern financial dashboard inspired by Stripe-style UI.

## Features

- Revenue progress bar
- KPI cards for Revenue, Expenses, Profit, and Cash
- MTD / QTD / YTD period toggle
- Revenue trend line chart
- Actual vs plan bar chart

## Run locally

1. Install Node.js 20+ (or 22 LTS).
2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000).

The dashboard currently uses built-in sample data and does not require any environment variables.

## Build for production

```bash
npm run build
npm run start
```
